# GitHub Pages Setup

## Quick Setup

Your interactive article is ready to be hosted on GitHub Pages!

## Enable GitHub Pages

1. Go to your repository on GitHub: https://github.com/SosoTughushi/ElectricSheep
2. Click on **Settings** (top menu)
3. Scroll down to **Pages** (left sidebar)
4. Under **Source**, select:
   - **Branch**: `master`
   - **Folder**: `/docs`
5. Click **Save**

## Access Your Article

Once GitHub Pages is enabled (may take 1-2 minutes), your article will be available at:

**https://sosotughushi.github.io/ElectricSheep/ai-first-repository-guide.html**

## Direct Link

You can also access it directly via:
- **https://sosotughushi.github.io/ElectricSheep/ai-first-repository-guide.html**

## Note

GitHub Pages may take a few minutes to build and deploy after enabling. If you see a 404 error, wait a few minutes and try again.
