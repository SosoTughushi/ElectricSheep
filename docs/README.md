# Electric Sheep Toolset - Documentation

This directory contains centralized documentation for the Electric Sheep toolset.

## Directory Structure

- **guides/**: User guides and tutorials
- **api/**: API documentation (when MCP is implemented)
- **architecture/**: Architecture and design documentation

## Tool-Specific Documentation

Each tool maintains its own documentation:
- See `tools/*/README.md` for tool-specific docs
- See `tools/*/MANIFEST.json` for tool metadata
- See `tools/*/docs/` for detailed tool documentation
