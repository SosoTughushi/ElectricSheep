# AI Article Composer

Orchestrator tool that combines text sections, diagrams, and complexity settings into final article.

## Overview

This tool performs the final composition step, combining all generated components (text sections, diagrams, complexity settings) into a complete article.

## Usage

```powershell
.\scripts\compose-article.ps1 -TextSections @(...) -OutputPath "./output/article.md"
```
