# AI Complexity Adapter

Modular tool for adjusting paragraph complexity for different audiences.

## Overview

This tool adapts content to three complexity levels (simple, medium, advanced) for use in hover UI. It's used by the AI Article Writer to create adaptive content.

## Usage

```powershell
.\scripts\adapt-complexity.ps1 -Content "Your content" -TargetLevel "simple"
```
