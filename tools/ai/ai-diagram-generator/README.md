# AI Diagram Generator

Modular tool for generating Mermaid diagrams from user descriptions.

## Overview

This tool creates Mermaid diagrams (flowcharts, sequence diagrams, etc.) from text descriptions. It's used by the AI Article Writer to add visualizations to articles.

## Usage

```powershell
.\scripts\generate-diagram.ps1 -Description "Your diagram description"
```
