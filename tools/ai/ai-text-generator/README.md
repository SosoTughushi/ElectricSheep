# AI Text Generator

Modular tool for generating article text sections from prompts.

## Overview

This tool generates text sections (paragraphs, headings, lists, code blocks) for use in articles. It's used by the AI Article Writer to create content.

## Usage

```powershell
.\scripts\generate-text.ps1 -Prompt "Your prompt here"
```
